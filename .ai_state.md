### Data/schema findings (2026-06-13)
- `TAP.confidence` is populated **only** by the whisper-v1 aligner (`teAutoTimingAndSwitch`), never by manual timing edits. Documented in CLAUDE.md.
- `arrow` field on DATA syllables is serialized in session JSON but absent in verse JSON files. Now documented in CLAUDE.md.

- `duration_s: 15.5` was a template sentinel in all 3 sample verses; replaced with `null` and schema updated to `["number","null"]`.

### Security findings (2026-06-13)
- Drive `_gdRenderList()` XSS: `f.name` was embedded in onclick= HTML attribute with only `'` escaping — backslash injection bypassed it. Fixed with `createElement` + `addEventListener`.
- `showMsg()` was missing `>` and `"` from its escape chain — potential XSS if caller ever passes external content.
- Firebase/Google API keys in client code are public-by-design — security is via Firestore rules + authorized domains, NOT key secrecy.
